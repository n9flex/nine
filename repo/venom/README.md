-- Overview --
Automated Metasploit exploit runner. Scans all open ports on a target, searches for matching exploits, auto-fills options, and chains them until a shell is opened.

-- Usage --
venom <ip>

-- Examples --
venom 192.168.1.1
venom 10.0.0.5

-- Behavior --
1. Scans all ports on the target subnet
2. Displays results: OPEN (green), FILTERED (yellow), CLOSED (red)
3. For each open port, searches Metasploit for matching exploits
4. Auto-sets RHOST, RPORT and Version from scan data
5. Prompts for any extra required options
6. Runs exploits until a shell is opened, then stops

-- Requirements --
- msfconsole (auto-installed if missing)
- sora library (https://github.com/repository?r=sora)
