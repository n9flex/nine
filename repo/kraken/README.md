-- Overview --
Fully automated mass WiFi password cracker for HackHub.

-- Usage --
kraken

-- How it works --
1. Finds monitor mode interface
2. Scans all WiFi networks
3. Attacks in batches of 10 concurrently:
   - Deauth to capture handshakes
   - Cracks with hashcat
4. Saves results to networks.txt in current directory
5. Cleans up pcap files from current directory

Files created in current directory
- .pcap files (temporary, auto-deleted)
- networks.txt (final results)

Output format (networks.txt)
NetworkName - password123
AnotherNet - [FAILED]

-- Requirements --
- apt-get install node
- sora library (https://github.com/repository?r=sora)
