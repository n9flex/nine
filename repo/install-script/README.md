--------------------------------------------------
Global Install Script
Author: Empasm
--------------------------------------------------

An automated installation script for registering global terminal shortcuts.
This tool is meant to help anyone publishing a multi-file script.

--- FEATURES ---
- Global Shortcut Registration: Creates a link in `/lib` to run your tool from anywhere.
- Config Management: Generates a `.dat` file for persistent settings.
- Conflict Resolution: Prompts for a new name if the command is already taken.
- Professional UI: Supports ASCII art and "fake" loading sequences for polish.
