-- Overview --
Recursive file search for a FileSystem API. Supports Linux-style flags and wildcards.

-- Usage --
find <pattern> [path] [options]

-- Options --
-r, --root           Search from /
-h, --help          Show help

-- Examples --
find "*.ts"
find "*.txt" /home
find "sora*" --root

-- Requirements --
- apt-get install node
- sora library (https://github.com/repository?r=sora)
